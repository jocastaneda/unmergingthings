WordPress posts content restriction.

== Description ==

You can use this plugin to show up notice on article / page. Like "Cookie Notice" or "Adult Content Notice".

**Plugin Features**

* Free for everyone.
* Show popup notice on post / author archive page.
* Insert advertisement to notice box.
* Custo notice box n title / content / button.
* Custom decline button URL.

== Installation ==

1. Upload WP Content Restriction to your blog.
1. Activate it.
1. Go Settings > WP Content Restriction to setup.
1. Go article editor or user option page enable restriction.
